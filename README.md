# 🚨 Women Safety App

An AI-powered Women Safety Application built using Streamlit to provide emergency protection, live location access, safety assistance, and evidence collection features for women.

This project was developed as a hackathon innovation project with a focus on personal security, emergency response, and smart safety technology.

---

# 🌟 Features

## 🚨 Emergency SOS Alert
- One-click SOS emergency activation
- Displays emergency alert instantly
- Generates Google Maps live location link
- Shows user details and emergency information

---

## 📍 Live Location Tracking
- Displays:
  - Current city
  - Latitude
  - Longitude
- Opens location directly in Google Maps

---

## 📸 Camera Evidence Capture
- Capture emergency photos instantly
- Useful for evidence collection during unsafe situations
- Uses Streamlit camera input feature

---

## 🎤 Secret Audio Recording
- Simulates hidden emergency recording
- Can be upgraded with real-time audio recording
- Useful for safety evidence

---

## 🚶 Safe Walk Timer
- User sets travel duration
- If timer expires:
  - Emergency alert is triggered
- Helpful during late-night travel

---

## 🧠 AI Stress Detection
- Simulates emotional stress analysis
- Detects:
  - High stress
  - Moderate stress
  - Safe emotional state
- Future-ready for real AI integration

---

## 🏥 Nearby Emergency Help
Quick access to:
- Nearby Police Stations
- Nearby Hospitals

Integrated using Google Maps search links.
